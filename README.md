# React Assessment

## Technologies used
- [React](https://v3.nuxtjs.org/) 
- [TailwindCSS](https://tailwindcss.com/)
- [SASS](https://sass-lang.com/)

## Build Setup
```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm start

# build for production
$ npm run build

```
